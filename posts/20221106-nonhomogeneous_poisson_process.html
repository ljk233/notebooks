Rat in a maze

(M343 Book 2, Example 6.5)

Let error rate, Callable, be a function that accepts the parameter t, float, and returns the expected rate of errors by a rat learning its way around a maze. A precondition is that t \(\ge\) 0.

def error_rate(t):
    import numpy as np
    return 8 * np.exp(-t)

Let rat maze, NonHomogenousEvents, be the non-homogenous Poisson process that models the occurrence of errors by a rat learning their way around a maze.

rat_maze = NonHomogenousEvents(error_rate)  # per hour

Return the expected numbers of errors in the first hour, second hour, and third hour.

ts = [0, 1, 2, 3]
for t1, t2 in zip(ts[:-1], ts[1:]):
    print(f'μ({t1}, {t2}) = {rat_maze.interval_rate(t1, t2): .4f}')
μ(0, 1) =  5.0570
μ(1, 2) =  1.8604
μ(2, 3) =  0.6844
Note

We use x again to return a model for the number of events in an interval of time, but note that it has a different signature to the Events and MultivariateEvents classes.

Return the probability that the rat makes more than two errors in the second hour.

rat_maze.x(1, 2).sf(2)
0.285589

Learning to ride a bike

(M343 Book 2, Activity 6.2)

Tip

We use a lambda function here to define the rate function.

Let bike accidents, NonHomogenousEvents, be the non-homogenous Poisson process that models the occurrence of bike accidents per day by a child learning to ride their bike.

bike_accidents = NonHomogenousEvents(lambda t: 24 / (2 + t))  # t >= 0

Return the expected number of accidents in the third week.

bike_accidents.interval_rate(0, 7)
36.097858

Return the probability that the girl has eight accidents in the third week.

bike_accidents.x(14, 21).pmf(8)
0.135497

Return the probability that the fourth week is free of accidents.

bike_accidents.x(21, 28).pmf(0)
0.001700

## Summary

This notebook covers part of M343 Book 2, Part 2, which introduces the Poisson process as a means to model random events that occur in time.

We could not find a Python module that modelled the different type of Poisson processes covered in M343, so we wrote poissonprocesses, a Python processes that handles Poisson processes as summarised by M343.

This notebook covers a **non-homogeneous Poisson process**, which is modelled by the `NonHomogeneousEvents` class.

## NonHomogeneousEvents

> `poissonprocess.time.NonHomogeneousEvents`

A class to model a non-homogeneous Poisson process.

### Attributes

*rates*, `Callable`:
a function that the interval rate at time *t*.

### Methods

> `interval_rate(t1: float | int, t2: float | int) -> float`

Return the interval rate in a bounded window of time, *t1* $\to$ *t2*.

Preconditions:

- *t1* < *t2*

> `x(t1: float | int, t2: float | int) -> st.rv_discrete`

Return a model for the number of events that occur ina bounded window of time, $X(t_{1}, t_{2}) \sim \text{Poisson} (\mu (t_{1}, t_{2}))$.

Preconditions:

- *t1* < *t2*
