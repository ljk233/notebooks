This notebook covers part of M343 Book 2, Part 2, which introduces the Poisson process as a means to model random events that occur in time.

Events

poissonprocess.time.Events

A class to model a Poisson process.

Attributes

rate, float | int: the rate of occurrence of the event, \(\mu\).

Properties

t, scipy.stats.rv_continuous: the model for the time between successive events, \(T \sim M(\mu)\).

Methods

x (t: float | int) -> scipy.stats.rv_discrete

Return the model for the number of events that occur in t units of time, \(X(t) \sim \text{Poisson} (\mu t)\).

Example

(M343 Book 2, Activity 3.2)

Import the model.

from poissonprocesses.time import Events

Lower the precision of the return floats.

%precision 6
'%.6f'

Let earthquakes, Events, be the Poisson process that models the incidences of major earthquakes worldwide.

earthquakes = Events(12/14)  # per year
Tip

Both method x and property t return SciPy random variables, so the usual methods can be called on the returned object.

Return the probability that there will be at least three major earthquakes in a period of ten years.

earthquakes.x(10).sf(2)
0.991228

Return the probability that the waiting time between successive major earthquakes exceeds two years.

earthquakes.t.sf(2)
0.180092

Return the median time between successive major earthquakes.

earthquakes.t.median()
0.808672

## Summary

This notebook covers part of M343 Book 2, Part 2, which introduces the Poisson process as a means to model random events that occur in time.

We could not find a Python module that modelled the different type of Poisson processes covered in M343, so we wrote poissonprocesses, a Python processes that handles Poisson processes as summarised by M343.

This notebook covers a single **Poisson process**, which is modelled by the `Events` class.

## Events

> `poissonprocess.time.Events`

A class to model a Poisson process.

### Attributes

*rate*, `float | int`:
the rate of occurrence of the event, $\mu$.

### Properties

*t*, `scipy.stats.rv_continuous`:
the model for the time between successive events,  $T \sim M(\mu)$.

### Methods

> `x (t: float | int) -> scipy.stats.rv_discrete`

Return the model for the number of events that occur in *t* units of time, $X(t) \sim \text{Poisson} (\mu t)$.

## Example

*(M343 Book 2, Activity 3.2)*

Import the model.

```{python}
from poissonprocesses.time import Events
```

Lower the precision of the return `float`s.

```{python}
%precision 6
```

Let earthquakes, `Events`, be the Poisson process that models the incidences of major earthquakes worldwide.

```{python}
earthquakes = Events(12/14)  # per year
```

:::{.callout-tip}
Both method `x` and property `t` return *SciPy* random variables, so the usual methods can be called on the returned object.
:::

Return the probability that there will be at least three major earthquakes in a period of ten years.

```{python}
earthquakes.x(10).sf(2)
```

Return the probability that the waiting time between successive major earthquakes exceeds two years.

```{python}
earthquakes.t.sf(2)
```

Return the median time between successive major earthquakes.

```{python}
earthquakes.t.median()
```
