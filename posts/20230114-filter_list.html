Problem

We have a some sequence that we would like to filter on some predicate.

Solution

We have a number of different options:

  1. Use the filter function,
  2. Use the pipe module.

Example

We would like to select all even numbers from a list of ints.

Let nums be an interator that when iterated over, returns

nums = range(1, 11)

Use filter()

Return the even numbers in nums.

def is_even(x): return x % 2 == 0
list(filter(is_even, nums))
[2, 4, 6, 8, 10]

Use pipe

We can use either filter or where in the pipe module.

Import the dependencies.

import pipe  # noqa

Select odd numbers from nums.

is_odd = pipe.where(lambda x: x % 2 != 0)
list(nums | is_odd)
[1, 3, 5, 7, 9]

Select numbers that are divisible by 3 from nums.

is_div_3 = pipe.filter(lambda x: x % 3 == 0)
list(nums | is_div_3)
[3, 6, 9]

## Problem

We have a some sequence that we would like to filter on some predicate.

## Solution

We have a number of different options:

1. Use the `filter` function,
2. Use the `pipe` module.

## Example

We would like to select all even numbers from a `list` of `int`s.

Let *nums* be an interator that when iterated over, returns 

```{python}
nums = range(1, 11)
```

### Use `filter()`

Return the even numbers in *nums*.

```{python}
def is_even(x): return x % 2 == 0
```

```{python}
list(filter(is_even, nums))
```

### Use `pipe`

We can use either `filter` or `where` in the `pipe` module.

Import the dependencies.

```{python}
import pipe  # noqa
```

Select odd numbers from *nums*.

```{python}
is_odd = pipe.where(lambda x: x % 2 != 0)
list(nums | is_odd)
```

Select numbers that are divisible by 3 from *nums*.

```{python}
is_div_3 = pipe.filter(lambda x: x % 3 == 0)
list(nums | is_div_3)
```
