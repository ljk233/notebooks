CompoundEvents

poissonprocess.time.CompoundEvents

A class to model a compound Poisson process.

Attributes and properties

outcomes, st.rv_discrete: a model for the different outcomes that can occur.

Preconditions:

  • t > 1

Items purchased

M343 Activity 7.1

Note

We use a user-defined discrete random variable in this example.

Let n items be a discrete random variable that models the number of items bought by a customer at a village shop.

n_items = st.rv_discrete(values=((0, 1, 2, 3),
                                 (0.2, 0.3, 0.4, 0.1)))

Initialise leaving customers, the compound Poisson process that models customers leaving a shop and the number of items they have purchased.

leaving_customers = CompoundEvents(n_items, 12)  # per hour
Tip

We can access the outcomes attributes to return the model for the outcomes of an individual event.

Return the mean and variance of the number of items bought by a customer leaving the shop.

# mean, variance
leaving_customers.describe_outcomes()
array([1.4 , 0.84])

Return the mean and variance of the total number of items bought by customers over a nine hour period.

leaving_customers.describe_total_outcomes(9)
array([151.2, 302.4])

Return the index of dispersion.

leaving_customers.index_of_dispersion
2.000000

## Summary

This notebook covers part of M343 Book 2, Part 2, which introduces the Poisson process as a means to model random events that occur in time.

We could not find a Python module that modelled the different type of Poisson processes covered in M343, so we wrote *poissonprocesses*, a Python processes that handles Poisson processes as summarised by M343.

This notebook covers a **compund Poisson process**, which is modelled by the `CompoundEvents` class.

## CompoundEvents

> `poissonprocess.time.CompoundEvents`

A class to model a compound Poisson process.

### Attributes and properties

*index_of_dispersion*, `float` (`Property`)

*outcomes*, `st.rv_discrete`:
a model for the different outcomes that can occur.

*rate*, `float | int`:
the rate of occurrence of the event, $\mu$.

### Methods

`describe_outcomes() -> np.array`

> Return the mean and variance of the outcome of an event.

`describe_total_outcomes(t: float | int) -> np.array`

> Return the mean and variance of $S$(*t*), the total number of of outcomes in *t* units of time.
>
> Preconditions:
>
> - *t* > 1

`total_mean(t: float | int) -> float`

> Return the expected value of $S$(*t*), the total number of of outcomes in *t* units of time.
>
> Preconditions:
>
> - *t* > 1

`total_var(t: float | int) -> float`

> Return the variance of $S$(*t*), the total number of of outcomes in *t* units of time.
>
> Preconditions:
>
> - *t* > 1
