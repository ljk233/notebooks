## Problem

We would like to model a **continuous random variable** (rv) that is distributed by a non-standard probability distribution.

## Solution

We model it declaring a new class that inherits from *SciPy's* `rv_continuous` class that overrides the `_pdf` method.

## Example

Let $T$ be a continuous rv that has p.d.f.,

$$
f(t) = \frac{t}{48}, \enspace 2 \le t \le 10.
$$

Import the dependencies.

```{python}
from scipy import stats as st
```

Declare a new subclass of `rv_continuous`, and override the `_pdf` method.

:::{.callout-note}
We override the constructor here because $T$ is bounded.

If it was unbounded, then we do not need to override the constructor.
:::

```{python}
class t_gen(st.rv_continuous):
    def __init__(self):
        super().__init__(a=2, b=10)

    def _pdf(self, t):
        return t / 48
```

Initialise *t*, an instance `t_gen`.

```{python}
t = t_gen()
```
