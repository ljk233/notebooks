Plot The Probability Density Function Of A Continuous Random Vaiable

Probability
Visualisation
Author

Liam Kelly

Published

January 14, 2023

Problem

We have a continuous random variable whose p.d.f. we would like to visualise.

Solution

We can use the fuction below to produce a line chart of the p.d.f.

Warning

This requires seaborn >= 0.12.

plot_pdf(rv: rv_continuous) -> so.Plot

import numpy as np
from scipy import stats as st
from seaborn import objects as so


def plot_pdf(rv) -> so.Plot:
    """Plot the approximate probability distribution of T, the waiting
    time between events.
    """
    xs = np.linspace(rv.ppf(0.005), rv.ppf(0.995), num=100)
    return(
        so.Plot(x=xs, y=rv.pdf(xs))
        .add(so.Line())
        .label(x='x', y='f(x)', title='Probability distribution')
    )

Examples

plot_pdf(st.norm())

plot_pdf(st.gamma(2, 5))
