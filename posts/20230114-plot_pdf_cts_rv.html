Problem

We have a continuous random variable whose p.d.f. we would like to visualise.

Solution

Import the dependencies.

import numpy as np
from scipy import stats as st
from seaborn import objects as so

Initialise the random variable.

Note

We use the normal distribution for this example, but it should work with any instance of rv_continuous.

x = st.norm()

Initialise xs, which represents the event space.

xs = np.linspace(x.ppf(0.01), x.ppf(0.99), num=100)

Plot the p.d.f.

(
    so.Plot(x=xs, y=x.pdf(xs))
    .add(so.Line())
    .label(x='X', y='f(x)', title='Probability distribution')
)

## Problem

We have a continuous random variable whose p.d.f. we would like to visualise.

## Solution

Import the dependencies.

```{python}
import numpy as np
from scipy import stats as st
from seaborn import objects as so
```

Initialise the random variable.

:::{.callout-note}
We use the normal distribution for this example, but it should work with any instance of `rv_continuous`.
:::

```{python}
x = st.norm()
```

Initialise *xs*, which represents the *event space*.

```{python}
xs = np.linspace(x.ppf(0.01), x.ppf(0.99), num=100)
```

Plot the p.d.f.

```{python}
(
    so.Plot(x=xs, y=x.pdf(xs))
    .add(so.Line())
    .label(x='X', y='f(x)', title='Probability distribution')
)
```
